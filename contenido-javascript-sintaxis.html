Sesión 2: Lógica y sintaxis base

1. Variables

  • Variables (let): Se usan para crear contenedores de información cuyo valor puede cambiar durante el programa.
  • Constantes (const): Sirven para guardar datos que deben permanecer fijos y no pueden ser sobrescritos.
    • Concepto Sintaxis Ejemplo
    Variables (Cambiantes) let let puntos = 10;
    Constantes (Fijas) const const pi = 3.14;

2. Datos simples

  • Texto (String): Representa secuencias de caracteres y siempre debe ir encerrado entre comillas dobles o simples.
  • Números (Number): Se utilizan para realizar cálculos matemáticos y se escriben directamente sin usar comillas.
  • Booleanos: Son valores de lógica simple que solo pueden ser verdaderos (true) o falsos (false).
  • Unir textos: Permite insertar variables dentro de una frase de forma sencilla usando las comillas invertidas.
    • Concepto Sintaxis Ejemplo
    Texto (String) " " o ' ' "Hola Mundo"
    Números (Number) Sin comillas 25
    Booleanos true / false let esVisible = true;
    Unir textos ${ } (Backticks) `Hola ${nombre}`

3. Estructuras

  • Listas (Arrays): Permiten almacenar múltiples valores de forma ordenada dentro de una sola variable.
  • Objetos: Estructuras que agrupan diferentes características y datos relacionados mediante claves y valores.
  • Comentario corto: Anula una sola línea de código para que el navegador no la lea, ideal para notas rápidas.
  • Comentario largo: Permite anular bloques completos de varias líneas de texto o código simultáneamente.
    • Concepto Sintaxis Ejemplo
    Listas (Arrays) [ ] ["pan", "leche"]
    Objetos { } { nombre: "Ana" }
    Comentario corto // // Esto es una nota
    Comentario largo /* */ /* Varias líneas */

4. Lógica

  • Salida de datos: La instrucción para mostrar cualquier valor o mensaje dentro de la consola del navegador.
  • Condicional: Evalúa si una prueba es verdadera para decidir si ejecuta o no un bloque de código.
    • Concepto Sintaxis Ejemplo
    Salida de datos console.log() console.log(edad);
    Condicional if (condición) { } if (x > 5) { ... }

5. Resumen

Para organizar la sintaxis de JavaScript, dividimos el contenido en cuatro bloques lógicos: primero, el almacenamiento mediante variables y constantes para guardar datos; segundo, los tipos de información básicos como textos, números y booleanos; tercero, la organización mediante listas (arrays) y objetos para manejar estructuras complejas; y cuarto, el control y salida a través de condicionales y la consola para ejecutar acciones. Esta estructura permite separar claramente qué sirve para guardar datos, qué define su naturaleza y qué controla el comportamiento del programa.

Tarea: Responde a las siguientes cuestiones:

  • ¿Cuál es la diferencia principal entre usar let y const al crear una variable?
  • ¿Cómo se representan los tipos de datos "String" y "Number" en el código?
  • ¿Para qué sirven los Arrays y qué símbolo los identifica?

Respuesta (ejemplo):

  • let permite que el valor de la variable cambie más adelante, mientras que const protege el dato para que permanezca fijo y no pueda ser modificado.
  • Los datos de tipo "String" (texto) se escriben siempre entre comillas, mientras que los "Number" (números) se escriben directamente sin ellas.
  • Los Arrays sirven para almacenar una lista de múltiples valores de forma ordenada dentro de una sola variable y se identifican por el uso de corchetes [ ].
