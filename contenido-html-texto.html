Sesión 3: Etiquetas de contenido en HTML: texto, listas y tablas

Las etiquetas de texto son fundamentales en HTML porque permiten organizar, estructurar y dar significado al contenido escrito dentro de una página web.

Gracias a estas etiquetas, los navegadores pueden diferenciar títulos, párrafos, enlaces o palabras importantes, mostrando cada elemento de forma adecuada.

El uso correcto de etiquetas de texto mejora la legibilidad de la página, facilita la navegación del usuario y ayuda a los motores de búsqueda a comprender mejor la información.

1. ¿Qué son las etiquetas de texto?

Las etiquetas de texto son elementos HTML que permiten definir distintos tipos de contenido textual.

Cada etiqueta tiene una función específica:

  • Crear títulos y subtítulos
  • Separar párrafos
  • Resaltar información importante
  • Mostrar texto en cursiva o negrita
  • Insertar enlaces

Estas etiquetas permiten que el contenido sea más claro, ordenado y visualmente comprensible.

2. Etiquetas de texto en HTML

Sin etiquetas de texto, una página web mostraría toda la información de forma desordenada.

Las etiquetas ayudan a:

  • Organizar el contenido por niveles
  • Separar ideas y conceptos
  • Mejorar la experiencia del usuario
  • Facilitar la accesibilidad
  • Optimizar el posicionamiento SEO

Por ello, dominar estas etiquetas es esencial para cualquier desarrollador web.

HTML utiliza diferentes etiquetas para mostrar texto con significado. Por ejemplo, los títulos ayudan a organizar el contenido y los párrafos permiten separar ideas. Además, existen etiquetas para crear listas ordenadas, listas desordenadas y tablas.

  • <h1> - título principal
  • <h2> - subtítulo de segundo nivel

  • <h5> - subtítulo de quinto nivel
  • <h6> - subtítulo de sexto nivel
  • <p> - crea un párrafo
  • <br> - inserta un salto de línea
  • <hr> - crea una línea horizontal divisoria
  • <a> - crea un enlace
  • <b> - texto en negrita visual
  • <strong> - texto importante
  • <i> - texto en cursiva visual
  • <em> - texto enfatizado
  • <mark> - resalta texto
  • <small> - texto más pequeño
  • <code> - muestra código

A continuación, realiza la actividad práctica para identificar correctamente las principales etiquetas de texto.

Tarea cooperativa mediante puzzle de Aronson: En grupos de 4, cada alumno trabaja una microtarea distinta. Primero se reúne con compañeros que tienen el mismo rol para convertirse en experto y después vuelve a su grupo base para explicar su parte y construir el producto final común.

  1. Experto 1 - Importancia de las etiquetas de texto: Explica cómo los títulos y párrafos ayudan a separar ideas, mejorar la lectura y facilitar que el contenido tenga sentido.
  2. Experto 2 - Principales etiquetas de texto: Crea un bloque con <h1>, <p>, <strong> y <em>, diferenciando entre estilo visual e importancia semántica.
  3. Experto 3 - Listas: Organiza una serie de pasos con <ol> y una lista de conceptos con <ul>. Explica cuándo conviene usar cada una.
  4. Experto 4 - Tablas: Construye una tabla sencilla con <table>, <tr>, <th> y <td> para comparar HTML, CSS y JavaScript.

Producto final del grupo: una pequeña práctica conjunta donde aparezcan integradas las cuatro aportaciones y cada miembro pueda justificar su decisión.
