Sesión 3: Etiquetas de texto en HTML

Las etiquetas de texto son fundamentales en HTML porque permiten organizar, estructurar y dar significado al contenido escrito dentro de una página web.

Gracias a estas etiquetas, los navegadores pueden diferenciar títulos, párrafos, enlaces o palabras importantes, mostrando cada elemento de forma adecuada.

El uso correcto de etiquetas de texto mejora la legibilidad de la página, facilita la navegación del usuario y ayuda a los motores de búsqueda a comprender mejor la información.

1. ¿Qué son las etiquetas de texto?

Las etiquetas de texto son elementos HTML que permiten definir distintos tipos de contenido textual.

Cada etiqueta tiene una función específica:

  • Crear títulos y subtítulos
  • Separar párrafos
  • Resaltar información importante
  • Mostrar texto en cursiva o negrita
  • Insertar enlaces

Estas etiquetas permiten que el contenido sea más claro, ordenado y visualmente comprensible.

2. Importancia de las etiquetas de texto

Sin etiquetas de texto, una página web mostraría toda la información de forma desordenada.

Las etiquetas ayudan a:

  • Organizar el contenido por niveles
  • Separar ideas y conceptos
  • Mejorar la experiencia del usuario
  • Facilitar la accesibilidad
  • Optimizar el posicionamiento SEO

Por ello, dominar estas etiquetas es esencial para cualquier desarrollador web.

3. Principales etiquetas de texto

HTML utiliza diferentes etiquetas para mostrar texto con significado. Por ejemplo, los títulos ayudan a organizar el contenido y los párrafos permiten separar ideas.

En la siguiente imagen puedes ver algunos ejemplos de etiquetas de texto:

En la imagen se muestran ejemplos de cómo funcionan las etiquetas de texto en HTML. A la izquierda aparece el código, y a la derecha el resultado visual dentro de la página web.

  • <h1> — crea un título principal grande
  • <p> — muestra un párrafo
  • <a> — crea un enlace
  • <i> — muestra texto en cursiva
  • <em> — enfatiza texto
  • <b> — muestra texto en negrita
  • <strong> — resalta texto importante

4. Diferencias entre etiquetas similares

Algunas etiquetas pueden parecer similares visualmente, pero tienen diferencias semánticas:

  • <b> solo cambia el estilo visual a negrita.
  • <strong> indica importancia semántica.
  • <i> aplica cursiva visual.
  • <em> añade énfasis semántico.

Esto significa que ciertas etiquetas no solo modifican la apariencia, sino también el significado del contenido.

5. Buenas prácticas

Para escribir contenido correctamente en HTML se recomienda:

  • Usar títulos de forma jerárquica
  • No abusar de negritas o cursivas
  • Utilizar <strong> para información importante
  • Separar claramente párrafos
  • Usar enlaces descriptivos

Estas prácticas mejoran tanto la organización como la accesibilidad del sitio.

6. Resumen y actividad

Las etiquetas de texto permiten estructurar la información escrita en HTML, mejorando su presentación, organización y significado.

Comprender estas etiquetas es esencial para crear páginas web claras y profesionales.
