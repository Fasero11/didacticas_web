Sesión 4: Enlaces e imágenes en HTML

Los enlaces y las imágenes son elementos esenciales en cualquier página web. Gracias a ellos, las páginas pueden conectarse entre sí, facilitar la navegación y mostrar contenido visual que mejora la experiencia del usuario.

Mientras que los enlaces permiten desplazarse entre páginas, secciones o recursos, las imágenes ayudan a representar información de forma visual y atractiva.

1. ¿Qué son los enlaces en HTML?

Los enlaces, también llamados hipervínculos, permiten conectar diferentes recursos dentro o fuera de una página web.

Se crean mediante la etiqueta <a>, utilizando el atributo href para indicar el destino.

Los enlaces son fundamentales para la navegación web, ya que permiten al usuario moverse fácilmente entre diferentes contenidos.

2. Tipos principales de enlaces

Existen varios tipos de enlaces según su función:

Tipo de enlace Ejemplo Uso
Externo <a href="https://google.com">Google</a> Ir a otra página web
Interno <a href="contacto.html">Contacto</a> Ir a otra página del mismo sitio
Email <a href="mailto:correo@email.com">Enviar correo</a> Abrir cliente de correo
Ancla <a href="#seccion">Ir a sección</a> Navegar dentro de la misma página

3. Enlaces internos y anclas

Los enlaces internos permiten desplazarse dentro de una misma página utilizando identificadores únicos llamados id.

Este sistema resulta muy útil para:

  • Índices de contenido
  • Menús de navegación
  • Páginas extensas
  • Documentación educativa

Ejemplo: 

<a href="#imagen">Ir a la imagen</a>

<h3 id="imagen">Mi imagen</h3>

Cuando el usuario hace clic, el navegador se desplaza automáticamente hasta la sección indicada.

A continuación, realiza la siguiente actividad práctica para comprobar si comprendes correctamente los diferentes tipos de enlaces en HTML y su uso dentro de páginas web.

Tarea: Completa los siguientes enlaces con la información correcta:

<!-- Enlace externo -->
<a href="__________">Ir a Google</a>

<!-- Enlace interno -->
<a href="__________">Ir a la página de contacto</a>

<!-- Enlace de correo -->
<a href="__________">Enviar un correo</a>

Respuesta:

<a href="contacto.html">Ir a la página de contacto</a>

<a href="mailto:correo@email.com">Enviar un correo</a>

4. Imágenes en HTML

Las imágenes permiten añadir contenido visual a una página web y se insertan mediante la etiqueta <img>.

Para funcionar correctamente, esta etiqueta necesita principalmente:

  • src — ruta o dirección de la imagen
  • alt — descripción alternativa

Ejemplo: 

<img src="imagen.jpg" alt="Descripción de la imagen">

El atributo alt es importante para la accesibilidad y para mostrar información si la imagen no se carga correctamente.

5. Navegación hacia imágenes

Las imágenes también pueden formar parte de sistemas de navegación interna, permitiendo al usuario desplazarse rápidamente hacia secciones específicas.

Por ejemplo:

Ir al ejemplo de imagen

Ejemplo de imagen

Este sistema resulta especialmente útil en páginas largas o educativas, donde el usuario necesita localizar rápidamente ejemplos o recursos específicos.

Ahora que conoces cómo insertar imágenes y crear navegación interna mediante anclas, completa esta actividad para practicar la integración de imágenes dentro de una página HTML.

Tarea:

  1. Inserta una imagen usando la etiqueta <img>.
  2. Añade un título con id para la imagen.
  3. Crea un enlace que lleve a la imagen.
<a href="#__________">Ir a la imagen</a>

<h3 id="__________">Mi imagen</h3>
<img src="__________" alt="__________">

Respuesta:

<a href="#imagen">Ir a la imagen</a>

<h3 id="imagen">Mi imagen</h3>
<img src="imagen.jpg" alt="Mi imagen">

6. Videos en HTML

Además de imágenes y enlaces, HTML permite insertar videos directamente dentro de una página web para enriquecer el contenido visual e interactivo.

Para ello se utiliza la etiqueta <video>, que permite reproducir archivos de video almacenados en el proyecto o provenientes de otras fuentes.

Los atributos más importantes son:

  • src — ruta del archivo de video
  • controls — muestra controles de reproducción
  • width — ancho del reproductor
  • height — alto del reproductor

Ejemplo de video almacenado dentro de la carpeta del proyecto: 


<video src="videos/mi-video.mp4" controls width="100%">
  Tu navegador no soporta videos.
</video>

También puede utilizarse la etiqueta <source> para ofrecer diferentes formatos: 


<video controls width="100%">
  <source src="videos/mi-video.mp4" type="video/mp4">
  <source src="videos/mi-video.webm" type="video/webm">
  Tu navegador no soporta videos.
</video>

Se recomienda guardar los videos en una carpeta específica como videos/ dentro del proyecto para mantener una mejor organización.

Ejemplo práctico de video insertado en la página:

Ahora realiza la siguiente actividad práctica para insertar correctamente un video en tu página web.

Tarea: Completa el siguiente código usando las etiquetas correctas:

  <_____ src="videos/mi-video.mp4" type="video/mp4">

Respuesta:
