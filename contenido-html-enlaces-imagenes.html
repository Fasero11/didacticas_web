Sesión 4: Enlaces, imágenes y videos

Los enlaces y las imágenes son elementos esenciales en cualquier página web. Gracias a ellos, las páginas pueden conectarse entre sí, facilitar la navegación y mostrar contenido visual que mejora la experiencia del usuario.

Mientras que los enlaces permiten desplazarse entre páginas, secciones o recursos, las imágenes ayudan a representar información de forma visual y atractiva.

1. ¿Qué son los enlaces en HTML?

Los enlaces, también llamados hipervínculos, permiten conectar diferentes recursos dentro o fuera de una página web.

Se crean mediante la etiqueta <a>, utilizando el atributo href para indicar el destino.

Los enlaces son fundamentales para la navegación web, ya que permiten al usuario moverse fácilmente entre diferentes contenidos.

2. Tipos principales de enlaces

Existen varios tipos de enlaces según su función:

Tipo de enlace Ejemplo Uso
Externo <a href="https://google.com">Google</a> Ir a otra página web
Interno <a href="contacto.html">Contacto</a> Ir a otra página del mismo sitio
Email <a href="mailto:correo@email.com">Enviar correo</a> Abrir cliente de correo
Ancla <a href="#seccion">Ir a sección</a> Navegar dentro de la misma página

3. Enlaces internos y anclas

Los enlaces internos permiten desplazarse dentro de una misma página utilizando identificadores únicos llamados id.

Este sistema resulta muy útil para:

  • Índices de contenido
  • Menús de navegación
  • Páginas extensas
  • Documentación educativa

Ejemplo: 

<a href="#imagen">Ir a la imagen</a>

<h3 id="imagen">Mi imagen</h3>

Cuando el usuario hace clic, el navegador se desplaza automáticamente hasta la sección indicada.

A continuación, realiza la siguiente actividad práctica para comprobar si comprendes correctamente los diferentes tipos de enlaces en HTML y su uso dentro de páginas web.

Tarea cooperativa mediante puzzle de Aronson: En grupos de 4, cada alumno trabaja una microtarea distinta. Primero se reúne con compañeros que tienen el mismo rol para convertirse en experto y después vuelve a su grupo base para explicar su parte y construir el producto final común.

  1. Experto 1 - Enlaces y sus tipos: Crea un enlace externo, uno interno y uno de correo usando <a> y el atributo href. Explica la diferencia entre ellos.
  2. Experto 2 - Enlaces internos y anclas: Añade un índice con enlaces que lleven a secciones de la misma página mediante identificadores id.
  3. Experto 3 - Imágenes y navegación: Inserta una imagen con src y alt. Después crea un enlace que lleve directamente a esa imagen o sección visual.
  4. Experto 4 - Videos en HTML: Inserta un vídeo con <video>, <source> y controls. Explica qué función cumple cada atributo o etiqueta.

Producto final del grupo: una pequeña práctica conjunta donde aparezcan integradas las cuatro aportaciones y cada miembro pueda justificar su decisión.
