Sesión 2: Selectores CSS

Los selectores permiten elegir qué elementos HTML serán modificados.

1. Selector de etiqueta


p {
  color: green;
}

2. Selector de clase


.texto {
  font-size: 24px;
}

3. Selector de ID


#titulo {
  color: purple;
}

Escribe un selector de clase llamado .caja


.caja {
  background: lightgray;
}
