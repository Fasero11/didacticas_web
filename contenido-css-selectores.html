Sesión 2: Selectores CSS

Los selectores CSS sirven para indicar qué elementos de una página queremos modificar. Gracias a ellos podemos aplicar estilos a todos los párrafos, a una clase concreta, a un elemento con un identificador único o a elementos que están dentro de otros.

1. Selector de etiqueta

El selector de etiqueta aplica estilos a todos los elementos HTML de ese tipo. Por ejemplo, si escribimos p, la regla afectará a todos los párrafos de la página. 

p {
  color: green;
}

Este párrafo se ve afectado por el selector de etiqueta.

Este también, porque también es un elemento <p>.

2. Selector de clase

El selector de clase se escribe con un punto delante del nombre. Es muy útil porque permite aplicar el mismo estilo a varios elementos sin importar qué etiqueta HTML utilicen. 

.destacado {
  background: #e8f0fe;
  color: #1a73e8;
}

Texto normal.

Texto con clase .destacado.

Caja con la misma clase .destacado.

3. Selector de ID

El selector de ID se escribe con una almohadilla. Se utiliza para seleccionar un único elemento concreto de la página. Es habitual usarlo para secciones importantes o elementos que no se repiten. 

#aviso {
  border-left: 5px solid orange;
}

Mensaje normal.

Este mensaje simula un elemento seleccionado por ID.

4. Selectores combinados

También podemos combinar selectores para aplicar estilos solo a elementos que cumplen una condición concreta. Por ejemplo, podemos modificar únicamente los enlaces que están dentro de un menú. 

.menu-ejemplo a {
  color: white;
  background: #1f1f1f;
}

5. Resumen visual

Selector Ejemplo Cuándo usarlo
Etiqueta p Para modificar todos los elementos de un mismo tipo.
Clase .tarjeta Para reutilizar estilos en varios elementos.
ID #principal Para modificar un elemento único.
Combinado nav a Para seleccionar elementos dentro de otros elementos.

Tarea cooperativa mediante puzzle de Aronson: En grupos de 4, diseñad una tarjeta de aviso para una página educativa. Cada alumno se especializa en un tipo de selector, lo practica con su grupo de expertos y luego vuelve al grupo base para integrar todas las reglas.

  1. Experto 1 - Selector de etiqueta: Aplica un estilo general a todos los párrafos de la tarjeta usando el selector p. Debe mejorar la lectura del texto.
  2. Experto 2 - Selector de clase: Crea una clase reutilizable .importante para destacar varios elementos con el mismo estilo.
  3. Experto 3 - Selector de ID: Usa un identificador #aviso-principal para dar un estilo único al aviso más importante de la página.
  4. Experto 4 - Selectores combinados: Aplica estilos solo a los enlaces que estén dentro de .menu-ejemplo, sin afectar al resto de enlaces de la página.

Producto final del grupo: una tarjeta donde se usen correctamente los cuatro tipos de selectores y cada miembro pueda explicar por qué su selector era necesario.

<!-- HTML -->
<section id="aviso-principal">
  <h2>Aviso importante</h2>
  <p>Este texto usa el estilo general de los párrafos.</p>
  <p class="importante">Este mensaje está destacado con una clase reutilizable.</p>

  <nav class="menu-ejemplo">
    <a href="#">Inicio</a>
    <a href="#">Contenido</a>
  </nav>

  <a href="#">Enlace exterior al menú</a>
</section>

/* CSS */
p {
  line-height: 1.6;
  color: #333;
}

.importante {
  background-color: yellow;
  font-weight: bold;
  padding: 8px;
}

#aviso-principal {
  border-left: 5px solid orange;
  background: #fff8e1;
  padding: 20px;
  border-radius: 12px;
}

.menu-ejemplo a {
  color: white;
  background: #1f1f1f;
  padding: 8px 12px;
  border-radius: 8px;
  text-decoration: none;
}
