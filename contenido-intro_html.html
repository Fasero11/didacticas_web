Tema 1: Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje básico que se utiliza para crear páginas web. Permite estructurar el contenido mediante etiquetas, indicando qué es un título, un párrafo, una imagen o un enlace.

1. ¿Qué es HTML?

HTML no es un lenguaje de programación, sino un lenguaje de marcado. Esto significa que no ejecuta lógica, sino que organiza la información dentro de una página web.

2. Estructura básica de un documento HTML

Un documento HTML tiene una estructura básica que permite al navegador entender y mostrar correctamente el contenido de una página web.

  1. <!DOCTYPE html> — indica que el documento es HTML5
    • Siempre va en la primera línea
    • Ayuda al navegador a interpretar correctamente el código
  2. <html> — contiene todo el documento
    • Es la raíz del documento
    • Todo el contenido está dentro de esta etiqueta
  3. <head> — incluye información no visible
    • No se muestra en la página
    • Contiene datos importantes para el navegador
    • Incluye el título de la página
    • Puede contener configuraciones y enlaces a estilos (CSS)
  4. <body> — contiene el contenido visible
    • Aquí va todo lo que ve el usuario
    • Textos, imágenes, enlaces, botones
    • Es la parte principal de la página

Dentro de la etiqueta <head> normalmente encontramos:

  • <title> — el título de la página
  • <meta> — configuraciones del documento
  • <link> — enlaces a archivos CSS

Dentro de la etiqueta <body> encontramos:

  • <h1> - <h6> — encabezados (títulos)
  • <p> — párrafos de texto
  • <img> — imágenes
  • <a> — enlaces

Ordena correctamente las siguientes partes:

  • <head>

3. Etiquetas de texto

HTML utiliza diferentes etiquetas para mostrar texto con significado. Por ejemplo, los títulos ayudan a organizar el contenido y los párrafos permiten separar ideas.

En la siguiente imagen puedes ver algunos ejemplos de etiquetas de texto:

En la imagen se muestran ejemplos de cómo funcionan las etiquetas de texto en HTML. A la izquierda aparece el código, y a la derecha el resultado en la página web.

  • <h1> crea un título grande
  • <p> muestra un párrafo de texto
  • <a> crea un enlace (link)
  • <i> muestra el texto en cursiva
  • <em> enfatiza el texto
  • <b> muestra el texto en negrita
  • <strong> resalta texto importante

Como puedes ver, cada etiqueta tiene una función específica y cambia la forma en que se muestra el contenido en la página.

4. Enlaces en HTML

Los enlaces permiten conectar páginas entre sí. Son fundamentales para la navegación en la web.

Tipo de enlace Ejemplo Uso
Externo <a href="https://google.com">Google</a> Ir a otra web
Interno <a href="contacto.html">Contacto</a> Ir a otra página del sitio
Email <a href="mailto:correo@email.com">Enviar correo</a> Abrir correo

Tarea: Completa los siguientes enlaces con la información correcta:

<!-- Enlace externo -->
<a href="__________">Ir a Google</a>

<!-- Enlace interno -->
<a href="__________">Ir a la página de contacto</a>

<!-- Enlace de correo -->
<a href="__________">Enviar un correo</a>

Respuesta:

<!-- Enlace externo -->
<a href="https://google.com">Ir a Google</a>

<!-- Enlace interno -->
<a href="contacto.html">Ir a la página de contacto</a>

<!-- Enlace de correo -->
<a href="mailto:correo@email.com">Enviar un correo</a>

5. Imágenes en HTML

Las imágenes se insertan en HTML mediante la etiqueta <img>. Esta etiqueta necesita al menos dos atributos: src (ruta de la imagen) y alt (texto alternativo que describe la imagen).

También es posible crear enlaces internos para navegar dentro de la misma página. Este tipo de enlaces se llaman anclas y permiten moverse rápidamente de una parte del documento a otra.

Para crear un enlace interno, usamos la etiqueta <a> con el atributo href="#id", donde id es el identificador de un elemento en la página.

Ir al ejemplo de imagen

Cuando el usuario hace clic en este enlace, el navegador no cambia de página, sino que se desplaza automáticamente hasta el elemento que tiene el atributo id="ejemplo-imagen".

Este método es muy útil para crear menús, índices o facilitar la navegación en páginas largas.

Ejemplo de imagen

<img src="imagen.jpg" alt="Mi imagen">
