Sesión 3: Box Model y espaciado

En CSS, casi todos los elementos se comportan como cajas. Cada caja está formada por cuatro partes: el contenido, el padding, el borde y el margen. Entender este modelo ayuda a controlar mejor la separación y el tamaño de los elementos en una página web.

1. Las partes del Box Model

margin
border
padding
contenido
.caja {
  margin: 20px;
  border: 3px solid #1a73e8;
  padding: 20px;
}

2. Margin: espacio exterior

El margin crea espacio fuera del elemento. Sirve para separar una caja de otros elementos cercanos.

Sin margen
Con margen
Otra caja

3. Padding: espacio interior

El padding crea espacio dentro del elemento, entre el contenido y el borde. Hace que el texto no quede pegado a los límites de la caja.

Sin padding
Con padding: el texto respira mejor

4. Border: borde de la caja

El border dibuja una línea alrededor del elemento. Puede cambiar de grosor, color y estilo.

solid
dashed
rounded

5. Box-sizing

La propiedad box-sizing permite controlar cómo se calcula el tamaño real de una caja. Con border-box, el ancho incluye contenido, padding y borde, por lo que suele ser más fácil diseñar layouts estables. 

* {
  box-sizing: border-box;
}
content-box
padding + borde aumentan el tamaño
border-box
todo cabe en el ancho definido

Tarea cooperativa mediante puzzle de Aronson: En grupos de 4, construid una tarjeta de contenido llamada .caja-producto. Cada alumno se especializa en una parte del Box Model, la contrasta con su grupo de expertos y después vuelve al grupo base para montar una caja final equilibrada.

  1. Experto 1 - Concepto de caja y margin: Explica qué partes forman una caja CSS y añade margin para separar la tarjeta de otros elementos.
  2. Experto 2 - Padding: Añade espacio interior para que el texto no quede pegado al borde. Justifica cómo cambia la legibilidad.
  3. Experto 3 - Border: Diseña el borde de la tarjeta modificando grosor, color, estilo y redondeo.
  4. Experto 4 - Box-sizing: Aplica box-sizing: border-box y explica por qué facilita controlar el tamaño real de la caja.

Producto final del grupo: una tarjeta donde se identifiquen claramente margen, relleno, borde y cálculo del tamaño.

<!-- HTML -->
<article class="caja-producto">
  <h2>Tarjeta con Box Model</h2>
  <p>Esta caja utiliza margen, padding, borde y box-sizing.</p>
</article>

/* CSS */
* {
  box-sizing: border-box;
}

.caja-producto {
  width: 320px;
  margin: 20px;
  padding: 15px;
  border: 2px solid #1a73e8;
  border-radius: 10px;
  background: #f4f8ff;
}

.caja-producto h2 {
  margin-top: 0;
}

/* Reparto de expertos:

   Experto 2: padding.
   Experto 3: border y border-radius.
   Experto 4: box-sizing. */
