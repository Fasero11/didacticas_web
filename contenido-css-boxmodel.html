Sesión 3: Box Model y espaciado

Todos los elementos HTML funcionan como cajas.

1. Margin

El margin crea espacio exterior.

2. Padding

El padding crea espacio interior.

3. Border

El border añade bordes alrededor del contenido. 


div {
  margin: 20px;
  padding: 10px;
  border: 2px solid black;
}

Añade padding de 15px a una caja.


.caja {
  padding: 15px;
}
