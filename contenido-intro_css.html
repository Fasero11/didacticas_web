Sesión 1: Introducción a CSS

CSS (Cascading Style Sheets) es el lenguaje utilizado para dar estilo y diseño visual a las páginas web.

1. ¿Qué es CSS?

CSS permite modificar colores, tamaños, márgenes, posiciones y otros aspectos visuales.

2. ¿Para qué sirve CSS?

  • Aplicar colores
  • Modificar tamaños
  • Crear diseños modernos
  • Adaptar páginas a móviles

3. Ejemplo básico


p {
  color: blue;
  font-size: 20px;
}

Tarea: Cambia el color de un título usando CSS.


h1 {
  color: red;
}
