Sesión 1: Introducción a CSS

CSS es el lenguaje que se utiliza para definir el aspecto visual de una página web. Mientras HTML organiza el contenido, CSS decide cómo se ve: colores, tamaños, márgenes, fuentes, distribución, sombras y adaptación a distintos dispositivos.

Comenzar

1. Cambiar colores

Una de las funciones más básicas de CSS es cambiar el color del texto, del fondo o de los botones. Esto ayuda a destacar información importante y a crear una identidad visual. 

h1 {
  color: #1a73e8;
}

2. Modificar tamaños

CSS también permite cambiar el tamaño de textos, cajas, imágenes o botones. Esto sirve para crear jerarquía visual: los títulos suelen ser más grandes que los párrafos, y los elementos importantes pueden ocupar más espacio. 

.titulo {
  font-size: 32px;
}

3. Crear cajas visuales

Con CSS se pueden transformar bloques simples en tarjetas, avisos, botones o secciones completas. Para ello se combinan propiedades como background, border, border-radius, padding y box-shadow. 

.tarjeta {
  background: white;
  border: 1px solid #e8eef7;
  border-radius: 16px;
  padding: 20px;
  box-shadow: 0 8px 24px rgba(0,0,0,0.08);
}

4. Separar contenido y diseño

Una ventaja importante de CSS es que permite mantener el contenido en el archivo HTML y el diseño en un archivo externo, normalmente llamado styles.css. Así, una misma hoja de estilos puede aplicarse a muchas páginas. 

<link rel="stylesheet" href="css/styles.css">
Idea clave: HTML responde a “qué contenido hay” y CSS responde a “cómo se muestra ese contenido”.

Tarea cooperativa mediante puzzle de Aronson: En grupos de 4, cread una pequeña sección web llamada .presentacion-css. Cada alumno se convierte primero en experto de una parte, la trabaja con compañeros de otros grupos que tengan el mismo rol y después vuelve a su grupo base para enseñar su parte y unir el resultado final.

  1. Experto 1 - ¿Qué es CSS? Redacta una explicación breve que compare una página HTML sin estilos con una página con CSS. Añade una clase .sin-css y otra .con-css para que la diferencia sea visible.
  2. Experto 2 - Colores: Define los colores principales de la sección. Crea estilos para un título, un texto destacado y un botón usando color y background-color.
  3. Experto 3 - Tamaños y cajas visuales: Ajusta el tamaño del título y transforma un bloque simple en una tarjeta usando font-size, padding, border-radius y box-shadow.
  4. Experto 4 - Separar contenido y diseño: Comprueba que el HTML enlaza correctamente con css/styles.css mediante <link> y explica por qué es mejor separar el contenido del diseño.

Producto final del grupo: una mini sección HTML con estilos CSS donde se vea claramente qué aporta CSS a una página web.

<!-- HTML -->
<section class="presentacion-css">
  <h2>CSS transforma una página web</h2>
  <p class="destacado-css">HTML organiza el contenido y CSS mejora su aspecto visual.</p>
  <div class="tarjeta-css">
    <p class="sin-css">Antes: bloque simple sin diseño.</p>
    <p class="con-css">Después: bloque con colores, espacio y sombra.</p>
  </div>
  <button class="boton-css">Ver ejemplo</button>
</section>

/* CSS */
.presentacion-css {
  background-color: #f4f8ff;
  padding: 24px;
  border-radius: 18px;
}

.presentacion-css h2 {
  color: #1a73e8;
  font-size: 32px;
  text-align: center;
}

.destacado-css {
  color: #174ea6;
  font-weight: bold;
}

.tarjeta-css {
  background: white;
  padding: 20px;
  border-radius: 14px;
  box-shadow: 0 8px 24px rgba(0,0,0,0.08);
}

.boton-css {
  background-color: #1a73e8;
  color: white;
  border: none;
  padding: 10px 16px;
  border-radius: 10px;
}
