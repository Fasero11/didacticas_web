Sesión 4: Flexbox y diseño responsive

Flexbox es un sistema de CSS pensado para organizar elementos en filas o columnas de forma sencilla. Es especialmente útil para crear menús, tarjetas, botones alineados y diseños que se adaptan mejor a distintos tamaños de pantalla.

1. display: flex

Al aplicar display: flex a un contenedor, sus elementos hijos dejan de colocarse como bloques independientes y pasan a organizarse en una misma línea por defecto. 

.contenedor {
  display: flex;
}

2. justify-content: alineación horizontal

La propiedad justify-content controla cómo se distribuyen los elementos en el eje principal. Puede centrarlos, separarlos o llevarlos al inicio o al final. 

.contenedor {
  display: flex;
  justify-content: space-between;
}

3. align-items: alineación vertical

La propiedad align-items controla la alineación en el eje contrario. Es muy útil cuando queremos centrar elementos verticalmente dentro de una caja. 

.contenedor {
  display: flex;
  align-items: center;
}

4. flex-wrap y diseño responsive

Con flex-wrap, los elementos pueden saltar a una nueva línea cuando no caben. Esto permite crear diseños más adaptables a pantallas pequeñas. 

.tarjetas {
  display: flex;
  flex-wrap: wrap;
  gap: 12px;
}

Tarea cooperativa mediante puzzle de Aronson: En grupos de 4, cread una fila de tarjetas responsive para mostrar contenidos de HTML, CSS y JavaScript. Cada alumno se especializa en una propiedad clave de Flexbox, la practica con su grupo de expertos y después enseña al grupo base cómo integrarla.

  1. Experto 1 - ¿Qué es Flexbox? Activa el sistema flexible con display: flex y explica cómo cambia la colocación de los elementos.
  2. Experto 2 - Alineación horizontal: Usa justify-content para centrar o distribuir las tarjetas en el eje principal.
  3. Experto 3 - Alineación vertical: Usa align-items para controlar la alineación en el eje contrario y mejorar la presentación.
  4. Experto 4 - Diseño responsive: Añade flex-wrap y gap para que las tarjetas pasen a otra línea cuando no quepan.

Producto final del grupo: un contenedor flexible, centrado, con separación entre tarjetas y adaptable a pantallas pequeñas.

<!-- HTML -->
<section class="contenedor-flex">
  <article>HTML</article>
  <article>CSS</article>
  <article>JavaScript</article>
</section>

/* CSS */
.contenedor-flex {
  display: flex;
  justify-content: center;
  align-items: stretch;
  flex-wrap: wrap;
  gap: 12px;
}

.contenedor-flex article {
  width: 180px;
  padding: 20px;
  border-radius: 14px;
  background: #f4f8ff;
  text-align: center;
}
