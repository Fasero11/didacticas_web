Sesión 4: Flexbox y diseño responsive

Flexbox es un sistema de diseño que facilita organizar elementos.

1. display:flex


.container {
  display: flex;
}

2. justify-content

Permite alinear elementos horizontalmente.

3. align-items

Permite alinear elementos verticalmente.

Crea un contenedor flex.


.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
